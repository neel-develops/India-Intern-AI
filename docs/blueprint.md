# **App Name**: Internship Aligner

## Core Features:

- Student Profile Creation: Students create profiles with personal info, skills, preferences and resume upload. Stores student profiles (excluding resume file data) using local storage.
- Internship Listings Display: Display available internships with key details.
- Internship Filtering: Allows users to filter internships based on domain, skills, and location.
- Smart Match Suggestions: AI suggests internships to students, and suggests students for internships, as 'smart matches'. A tool analyzes student profiles and internship requirements to provide match suggestions.
- Eligibility Criteria: Internship Eligibility Criteria: Age 21-24 years, Not Employed Full Time, Not Enrolled Full Time, Family income less than 8 Lakhs PA, No Member has a Govt. Job, 12 months real-life experience in India's top companies

## Style Guidelines:

- Primary color: White (#FFFFFF) for a clean and modern look.
- Secondary color: Light Blue (#E0F2F7).
- Accent color: Orange (#FFB347) for CTAs and highlights.
- Body and headline font: 'Poppins' for a contemporary, precise feel.
- Simple, outline-style icons for key functions like filtering and profile management.
- Clean, card-based layout to present internships and student profiles.
- Subtle transitions and loading animations to enhance user experience.
